About BGPGuard

Project Overview

BGPGuard is a BGP anomaly detection platform that integrates data collection, feature extraction, label generation, data partitioning, and machine learning classification into a unified web-based interface.

Technology Stack

The platform leverages Flask for the backend, Bootstrap for the frontend, and PyTorch for deep learning models including LSTM, GRU, and VFBLS architectures. Real-time data is collected from RIPE and Route Views collection sites.

Capabilities

  • Real-time BGP anomaly detection with live visualization
  • Offline classification with customizable experiment parameters
  • Support for multiple ML architectures (LSTM, GRU, VFBLS)
  • Automated data download, feature extraction, and labeling pipeline
